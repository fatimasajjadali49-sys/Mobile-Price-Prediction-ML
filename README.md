#  Mobile Price Prediction using Machine Learning

##  Overview
This project predicts mobile phone prices based on specifications like RAM, storage, battery, camera, and more using machine learning models.

---

##  Objective
To build a regression model that estimates mobile prices using device features.

---

##  Dataset Features
- Brand, Model  
- RAM, Storage  
- Battery, Camera  
- Screen Size, OS  
- Release Year  
- Price (Target)

---

##  Data Processing
- Handled outliers using IQR method  
- Applied One-Hot Encoding for categorical data  
- Scaled features using StandardScaler  

---

##  Models Used
- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  

---

##  Results (R² Score)
- Linear Regression: 0.82  
- Decision Tree: 0.56  
- Random Forest: 0.39  

---

##  Best Model
Linear Regression performed best with highest accuracy.

---

##  Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

---

##  Note
I learned data cleaning, feature engineering, model training, and how different features affect mobile price prediction.
